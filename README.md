# GitHub Workshop Assessment

## Project Description
This project is created as part of a GitHub workshop assessment.
It demonstrates repository creation, file management, branching,
merging, and pull request workflows using GitHub.

## How to Run
1. Open the repository in a web browser.
2. Clone the repository using the command:
   git clone <repository-url>
3. Navigate to the project folder.
4. Open the files using any text editor.
